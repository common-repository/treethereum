== Description ==

treethereum Wordpress plugin can be used to add a direct link to buy a treethereum tree directly from the contract with your address automatically added to the transaction data field, adding new trees as your root.


* Insert a [treethereumplugin] shortcode wherever you like
* This plugin uses MetaMask to safely perform the sale operation
* It will show user a link to the Metamask site if the user doesnâ€™t have the Metamask installed
* We use a https://etherscan.io API to provide your client automatic rate calculations to USD

See the official site to see how it looks. https://www.treethereum.com/join/ "The treethereum Wordpress plugin")



== Installation ==

Enter your settings in admin pages and place the `[treethereumplugin]` shortcode wherever you need it.
